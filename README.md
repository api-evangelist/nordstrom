# Nordstrom (nordstrom)
Nordstrom is a leading fashion retailer offering compelling clothing, shoes, and accessories for men, women, and children through Nordstrom and Nordstrom Rack stores and online. Nordstrom does not currently operate a public developer portal or expose general-purpose APIs to third-party developers; this repository captures the company profile and any future developer-facing artifacts.
